# 💻 Computer Carrier Education (CCE) - Amethi 🚀

Yo! Welcome to the official repository of the **Computer Carrier Education** website. 
I built this project to give our local IT coaching institute a slick, ultra-modern, and premium online presence. No boring old-school templates—just pure custom design, glassmorphism, and smooth animations.

✨ **Live Vibe Check:** *(Add your deployed link here later)*

## 🎨 What's Inside?
This isn't just a basic static page; it's an experience. Here is what I baked into it:
- **Premium UI/UX:** Deep indigo and pink accents, glowing text gradients, and floating background orbs (yeah, those futuristic aesthetic blobs).
- **Glassmorphism:** Frosted glass cards with a sick 3D mouse-tracking tilt effect. Hover over the courses and watch them follow your cursor!
- **Smooth Animations:** Bouncy scroll reveals, continuous testimonial marquees, and dynamic button sweep effects.
- **Direct WhatsApp Integration:** No clunky databases needed for basic queries. The contact form directly connects students to the institute's WhatsApp with their pre-filled details.

## 🛠️ Tech Stack
Kept it clean and lightning-fast. No bloated frameworks, just pure code:
- **HTML5** (Semantic & Accessible)
- **CSS3** (Custom Properties, Keyframes, Flex/Grid, Glassmorphism)
- **Vanilla JavaScript** (DOM Manipulation, 3D Math for Tilt, Scroll Observers)
- **Node.js & Express** (For local serving and routing)

## 🚀 How to Run Locally 
Wanna spin this up on your machine? Easy.

1. **Clone this repo:**
   ```bash
   git clone https://github.com/yourusername/cce-amethi-website.git
   ```
2. **Navigate to the folder:**
   ```bash
   cd "COCHING WEBSIDE"
   ```
3. **Install dependencies** (just Express & CORS):
   ```bash
   npm install
   ```
4. **Start the local server:**
   ```bash
   node server.js
   ```
5. Open your browser and hit up `http://localhost:3000`. Boom. 💥

*(Alternatively, you can just use VS Code's "Live Server" extension on `index.html`!)*
